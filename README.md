# 🍬 SugarSweet – Diabetes Management Mobile App

SugarSweet (SS) is an Android application designed to support diabetic users in monitoring and managing their daily health activities.

The application provides tools for tracking blood sugar levels, meals, exercise, and generating useful statistics, while also supporting administrative management of system data.

## 📌 Overview

---

SugarSweet is a mobile information system that focuses on:

- Blood glucose monitoring
- Meal tracking
- Exercise logging
- Statistical analysis of health data
- Administrative management of system resources

The system is designed with two main user roles:
- **Diabetic User**
- **Administrator**

---

## 🧩 Core Features

### 👤 Diabetic User
- 🔐 Login & personal profile management
- 🩸 Record blood sugar measurements
- 🍽️ Log meals and ingredients
- 🏃 Track physical activity
- 📊 View statistics:
  - Daily / Weekly / Monthly trends
  - Deviations and patterns

---

### 🛠️ Administrator
- Manage exercises
- Manage ingredients
- Maintain system data
- Control application content

---

## 🏗️ System Architecture

The application follows a structured design based on:

- Activity-based Android architecture
- Separation of roles (Admin / User)
- Modular UI components
- Data-driven interaction between components
